■  フォント紹介  ■
　ダウンロードありがとうございます。
　ちょっぴりかわいいフォントを作りたいと思い作成しました。
　「ゆたココ」のプロポーショナルフォントバージョンです。
　作成にはTTEdit を使用して作成してOTEditで修正しています。
　多くの文字は手書きをして修正したものです。

特徴
　平仮名、片仮名の濁音、半濁音の区別がはっきりわかります。
　漢数字「〇」と、丸印「○」の区別も付きます。

注意
　Windows98・WindowsMe・では
　ATM Light for Windows（Adobe Type Manager Light for Windows）のインストール
　が必要になります。
　Adobe Type Manager Light
http://www.adobe.com/jp/products/atmlight/index.html


■　インストール　■
　Windowsの場合解凍した後でwindowsの「fonts」フォルダに放り込んで下さい。
　（ file://C:\WINDOWS\Fonts ）
　（古いバージョンがある場合は古いバージョンを削除して下さい。）
　Boldにはパスワードを掛けてありますので、
　お支払い後に届くパスワードをご入力下さい


■　アンインストール　■
　同フォントが選択、表示されていない状態でファイルを削除して下さい。
